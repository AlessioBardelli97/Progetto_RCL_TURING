- Nella cartella "Client" sono contenuti tutti i sorgenti java necessari per l'esecuzione del client.

- Nella cartella "ClientInput" sono contenuti i 3 file di input utilizzati dai client per i test.

- Nella cartella "Log" sono creati i file di l'output, del client e del server, dei vari test.

- Nella cartella "Server" sono contenuti tutti i sorgenti java necessari per l'esecuzione del Server.

- Nella cartella "Test" sono contenuti i 2 script per testare le funzionalità dell'applicazione TURING.
